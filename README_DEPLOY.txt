1) Загрузите все файлы и папки из этого архива в корневую директорию поддомена, например в /lab
2) В настройках сайта укажите точку входа в веб-приложение:
   wsgi:app
3) Во вкладке приложения или через консоль установите зависимости из requirements.txt
4) Добавьте переменные окружения:
   OPENAI_API_KEY=ваш_секретный_ключ
   APP_SECRET=длинная_случайная_строка
   OPENAI_MODEL=gpt-5-mini
5) Перезапустите приложение
6) Откройте сайт и проверьте /api/health

Полезные служебные адреса:
- Список сессий:
  /api/admin/sessions?secret=ВАШ_APP_SECRET
- Детали сессии:
  /api/admin/session/ID_СЕССИИ?secret=ВАШ_APP_SECRET
